# AI Portfolio

Mini Astro site for Diego Sierra's AI-focused work.

## Purpose

This subsite complements the main portfolio with a space for:

- AI-enabled SDLC experiments
- LLM platform patterns such as routing, caching, and fallbacks
- internal developer tooling, agents, plugins, and commands
- future case studies and architecture notes

## Commands

```bash
npm install
npm run dev
npm run build
```

The production build is copied into the root site's GitHub Pages artifact during the main deployment workflow.
