Privacy

Summary

This site is a lightweight personal portfolio hosted on GitHub Pages. I do not intentionally run analytics, ads, tracking pixels, or marketing cookies on the main site.

Hosting and delivery

The site is hosted on GitHub Pages, which may process standard server-level information such as IP address, user agent, and request metadata to deliver the site and keep the platform secure.

You can review GitHub's privacy information at GitHub Privacy Statement.

External services and links

Some links on this site point to third-party services such as LinkedIn, GitHub, Stack Overflow, Medium, Spotify, Calendly, and GitHub Sponsors. If you click those links, those services may collect data under their own privacy policies.

The Calendly button loads Calendly's widget assets so visitors can book a call. Calendly may process technical data needed to render the widget and schedule meetings.

Contact

If you need anything removed or want to ask about data related to this site, contact me through LinkedIn or GitHub.
